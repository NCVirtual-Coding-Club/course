# Add Key-Value Pairs to JavaScript Objects

## Final Code

{% highlight JavaScript %}
let foods = {
  apples: 25,
  oranges: 32,
  plums: 28
};

// Only change code below this line
foods.bananas = 13;
foods.grapes = 35;
foods.strawberries = 27;
// Only change code above this line

console.log(foods);
{% endhighlight %}
