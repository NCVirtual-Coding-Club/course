# Compound Assignment With Augmented Multiplication
## Summary
- The `*=` operator multiplies a variable by a number.

## Final Code

{% highlight JavaScript %}
var a = 5;
var b = 12;
var c = 4.6;

// Only change code below this line
a*=5;
b*=3;
c*=10;
{% endhighlight %}
