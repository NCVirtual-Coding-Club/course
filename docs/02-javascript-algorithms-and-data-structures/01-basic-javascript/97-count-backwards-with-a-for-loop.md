# Count Backwards With a For Loop
## Summary
- A `for` loop can count backwards.

## Final Code

{% highlight JavaScript %}
// Setup
var myArray = [];

// Only change code below this line
for (var i = 9; i >= 1; i-=2) {
  myArray.push(i);
}
{% endhighlight %}
