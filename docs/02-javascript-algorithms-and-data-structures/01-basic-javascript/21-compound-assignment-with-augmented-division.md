# Compound Assignment With Augmented Division
## Summary
- The `/=` operator divides a variable by another number.

## Final Code

{% highlight JavaScript %}
var a = 48;
var b = 108;
var c = 33;

// Only change code below this line
a/=12;
b/=4;
c/=11;
{% endhighlight %}
