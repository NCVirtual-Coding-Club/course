# Compound Assignment With Augmented Subtraction
## Summary
- The `-=` operator subtracts a variable by a number.

## Final Code

{% highlight JavaScript %}
var a = 11;
var b = 9;
var c = 3;

// Only change code below this line
a-=6;
b-=15;
c-=1;
{% endhighlight %}
