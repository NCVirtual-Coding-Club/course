# Appending Variables to Strings
## Summary
- Variables can be appended using the `+=` operator.

## Final Code

{% highlight JavaScript %}
// Change code below this line

var someAdjective = "fun";
var myStr = "Learning to code is ";
myStr+=someAdjective;
{% endhighlight %}
