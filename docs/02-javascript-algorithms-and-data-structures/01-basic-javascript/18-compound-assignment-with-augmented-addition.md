# Compound Assignment With Augmented Addition
## Summary
- It is common to use assignments to modify the contents of a variable.
- This can be done by using `+=`

## Final Code

{% highlight JavaScript %}
var a = 3;
var b = 17;
var c = 12;

// Only change code below this line
a+=12;
b+=9;
c+=7;
{% endhighlight %}
