# Inherit Styles from the Body Element
## Summary
- The `body` element can be styled using CSS.

## Final Code

{% highlight HTML %}
<style>
  body {
    background-color: black;
    color: green;
    font-family: monospace;
  }

</style>

<h1>Hello World</h1>
{% endhighlight %}
