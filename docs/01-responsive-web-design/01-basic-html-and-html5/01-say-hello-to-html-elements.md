# Say Hello to HTML Elements
## Summary
- Most HTML elements have an opening tag and an closing tag.
- Opening tags look like `<h1>` while closing tags look like `</h1>`.

## Final Code

{% highlight HTML %}
<h1>Hello World</h1>
{% endhighlight %}
