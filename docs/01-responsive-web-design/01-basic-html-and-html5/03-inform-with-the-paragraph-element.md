# Inform with the Paragraph Element
## Summary
- `p` elements are used for paragraphs.
- Use `<p>` (opening tag) and `</p>` (closing tag) to create a paragraph element.

## Final Code

{% highlight HTML %}
<h1>Hello World</h1>
<h2>CatPhotoApp</h2>
<p>Hello Paragraph</p>
{% endhighlight %}
