# Set the font-size of Paragraph Text
## Summary
- The `font-size` property can be applied to any element containing text.

## Final Code

{% highlight HTML %}
<style>
  p {
    font-size: 16px;
  }
</style>
<p>
  Lorem ipsum dolor sit amet...
</p>
{% endhighlight %}
