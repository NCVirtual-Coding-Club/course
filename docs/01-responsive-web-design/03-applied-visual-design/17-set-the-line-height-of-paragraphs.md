# Set the line-height of Paragraphs
## Summary
- The `line-height` property changes the amount of vertical space that each line of text gets.

## Final Code

{% highlight HTML %}
<style>
  p {
    font-size: 16px;
    line-height: 25px;
  }
</style>
<p>
  Lorem ipsum dolor sit amet...
</p>
{% endhighlight %}
